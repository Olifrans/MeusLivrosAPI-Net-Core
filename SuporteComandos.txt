Adcionando projeto no git apartir do Visual Studio

1-Clicar com o botão direito no solution
2-Selecione (Open Folder in File Explorer)
3-Na janela do Explore do Windows na barra de endereço digite (cmd)
5- Vai se abrir o CMD explorer
6- No cmd digite os comandos abaixos:

git init
git add .

7- Adcione os comando gerado pela criação do repositório:
…or create a new repository on the command line
echo "# MeusLivrosAspNetCoreWebApi" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/Olifrans/MeusLivrosAspNetCoreWebApi.git
git push -u origin main
…or push an existing repository from the command line
git remote add origin https://github.com/Olifrans/MeusLivrosAspNetCoreWebApi.git
git branch -M main
git push -u origin main
